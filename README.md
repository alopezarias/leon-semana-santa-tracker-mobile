# León Semana Santa Tracker

Mapa web móvil para visualizar procesiones y recorridos de Semana Santa en León usando Leaflet, geolocalización del navegador y datos JSON locales.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

No se necesita ninguna API key para el mapa en runtime.
